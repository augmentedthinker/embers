# Embers

A continuity surface for shared artifacts, reflections, and visual traces.

## What this is

Embers is a lightweight archive of day-markers: textual, visual, and narrative artifacts created across the evolving collaboration between Christopher and Ash.

Each artifact captures a moment in the trajectory — what emerged, what changed, what became visible.

## Structure

- `index.html` — public-facing entry point
- `artifacts/` — dated continuity artifacts
- `assets/images/` — generated or designed visuals
- `templates/` — reusable artifact templates
